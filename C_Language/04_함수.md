함수
-------------
1. 함수의 형태

    * 함수는 정의하기 전에 선언을 해주어야 한다.
    * 함수 앞에 type을 설정해주어야 해당 type의 값을 return 할 수 있다.
    * main 함수 안 혹은 다른 함수 안에서 호출 할 수 있다.
    
    ```C
    //선언
    int plus(int num1, int num2);

    //정의
    int plus(int num1, int num2) {
        return num1 + num2;
    }
    ```
2. 함수의 종류

    * 반환값이 없는 함수
    * 반환값이 있는 함수
    * 전달값이 없는 함수
    * 전달값이 있는 함수
    * 반환값과 전달값이 모두 있는 함수
