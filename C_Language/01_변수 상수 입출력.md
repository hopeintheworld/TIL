변수, 상수
-------------
1. 정수형 변수
    int age = 12; // 정수형 변수 선언.
    printf("%d\n", age);

2. 실수형 변수
    float f = 46.5 // 실수형 변수 선언
    printf("%.2f\n", f);

    double d = 4.428;
	printf("%.2f\n", d);

3. 상수 : 변하지 않는 수
    const int YEAR = 2000; // 변수 타입 앞에 const 추가

4. 문자(한 글자)
    char c = 'A';
    printf("%c\n", c);

5. 문자열
    char str[256];
    printf("%s\n", str);
