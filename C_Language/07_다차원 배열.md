다차원 배열
-------------
1. 다차원 배열 기본

    ```C
    // 2차원 배열
    int arr[2][5];
    //□□□□□
    //□□□□□

    int arr[3][3][3];
    ```

2. 다차원 배열 접근

    ```C
    arr[0][1];
    arr[1][4];
    ```
