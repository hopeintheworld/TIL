nodemon 사용법
--------------
* nodemon을 사용하면 코드에 변경사항이 생길 때마다 자동으로 이를 감지하여 코드를 재실행 해준다.
* 설치법
    * -D 옵션: devDependencies에 모듈이 추가되며, 개발이 완료되면 제거될 것이라는 의미.

    ```javascript
    npm i -g -D nodemon
    ```
